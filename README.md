# axe-cli.com

Marketing website for [AXe](https://github.com/cameroncooke/AXe) — a CLI tool for automating iOS Simulators using Apple's Accessibility APIs.

## Tech Stack

- [Next.js 15](https://nextjs.org/) with App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```
